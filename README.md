# Arduino-LEDs-Sequence

## Contextualization
Here we have 6 LEDs of different colors turning on and off sequentially. Each LED is connected to a pin ranging from pin 13 to pin 8

## Components:
- 1 - Arduino Uno R3
- 1 - USB Cable 
- 13 - Jumpers(male-to-male)
- 1 - ProtoBoard
- 6 - LED
- 6 - 220 Î© Resistor
